# Research Skill

This skill enables comprehensive web research and information gathering.

## Capabilities

### Web Search
Use the built-in WebSearch tool to find information on any topic.

When asked to research something:
1. Use WebSearch to find relevant sources
2. Read and analyze the results
3. Summarize key findings
4. Cite sources properly

### Best Practices

1. **Start Broad**: Begin with general search terms, then narrow down
2. **Multiple Sources**: Consult at least 3-5 sources for comprehensive research
3. **Verify Information**: Cross-reference facts across multiple sources
4. **Recent Information**: Prioritize recent sources for time-sensitive topics
5. **Summarize Clearly**: Present findings in organized, digestible format

### Output Format

Structure research results as:

```markdown
## Research Summary: [Topic]

### Key Findings
- Finding 1
- Finding 2
- Finding 3

### Detailed Analysis
[Comprehensive analysis of the topic]

### Sources
1. [Source 1 Title](URL)
2. [Source 2 Title](URL)
3. [Source 3 Title](URL)

### Recommendations
[If applicable, actionable recommendations based on research]
```

## When to Use
- When asked to research a topic or concept
- When gathering information before making decisions
- When summarizing articles or web content
- When fact-checking or verifying information
- When competitive analysis or market research is needed
